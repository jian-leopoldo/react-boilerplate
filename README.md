# react-boilerplate

Um boilerplate completo usando React + Redux, para você trabalhar em seus projetos

## Batteries

 - [x] React
 - [x] Redux
 - [x] Webpack
 - [x] Babel
 - [x] LESS
 - [ ] Relay
 - [ ] Immutable.js
 - [ ] Redux Simple Router
 - [ ] Redux Devtools
 - [ ] Jest / Mocha

## Setup

```bash
$ npm install
```

## Running

```bash
$ npm start
```
